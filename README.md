# Sistem Informasi Pegawai BNN Kabupaten Blitar

## Deskripsi
Aplikasi berbasis web untuk mengelola data pegawai  menggunakan Laravel dan MySQL.

## Teknologi
- Laravel 11
- PHP
- MySQL
- Laragon

## Fitur
- Login
- Dashboard
- CRUD Pegawai
- Data Pendidikan
- Data Kepangkatan
- Data Pengalaman
- Data Keluarga
- Upload Foto
